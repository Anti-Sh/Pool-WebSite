Проверка при регистрации:
- Поля не пустые;
- Пароли совпадают;
- Длина пароля 6 и более символов;
- E-mail ранее не был зарегистрирован;

Стоимость абонемента расчитывается по следующему принципу:
- Чем больше срок, тем меньше стоимость;
- Чем больше дней в неделю, тем больше стоимость;
- Абонемент на Спортивные секции дороже, чем обыкновенный абонемент;


Абонементы именые (создаются на абонентов)
Один пользователь может создать несколько абонентов
Один абонент может иметь несколько абонементов
